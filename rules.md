# 📚 서버 규칙

{% hint style="info" %}
* 서버 규칙은 수정 및 삭제될 수 있습니다.
* 게시되지 않은 규칙이더라도, 관리자의 재량에 의해 처벌될 수 있습니다.
* 서버 규칙 미숙지로 인하여 발생한 불이익은 책임지지 않습니다.
* 경고가 총 3회 누적될 시, 서버 접속이 차단됩니다.
* 경고 횟수는 상황, 심각성에 따라 정해집니다.
* 버그 악용 및 허점을 이용해 과도한 이익을 취할 경우 전부 밴 사유에 해당됩니다.
{% endhint %}

{% hint style="info" %}
* 마지막 경고 부여 이후 2주간 추가로 부여된 경고가 없을 때, 차감 요청을 하실 수 있습니다.
* 단, 처벌 사유에 따라 경고 차감이 불가능할 수 있습니다.
{% endhint %}
